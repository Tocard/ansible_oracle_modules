Role Name
=========

This role prepares partitions on disks to be members of Oracle ASM disk group.
Also installs Oracle ASM specific udev rules.

Requirements
------------

ASM Disks have names: /dev/nvme2 - /dev/nvme9
See: udevasm.sh

--------------

NONE

Dependencies
------------

NONE
